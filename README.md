## 1) ZoneTransefereDump.sh

This script queries the NS records of the specified target then tryies to dump the zone transefere files from each server.

### Requirments : "host" utality which is in most OSs by default.

Instalation : 
Downlad the script :

```
curl -O https://raw.githubusercontent.com/0x-miro/CustomBashScripts/main/ZoneTransefereDump.sh
```

Give it excutable permission :

```
 sudo chmod +x ./ZoneTransefereDump.sh
```

Usage : 

```
./ZoneTransefereDump.sh <Domain name>
```
